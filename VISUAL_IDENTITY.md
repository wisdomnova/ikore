## Accessibility Standards

### Color Contrast
- **Text on #61af50:** #ffffff (WCAG AAA)
- **Text on #ffffff:** #333333 (WCAG AAA)
- **Text on #000000:** #ffffff (WCAG AAA)

### Focus States
- **All interactive elements:** 2px solid #61af50 outline
- **Outline offset:** 2px

### Typography Minimums
- **Body text:** Minimum 16px for readability
- **Line height:** Minimum 1.5 for accessibility

---

## Image Guidelines

### Aspect Ratios
- **Hero Banner:** 16:9
- **Project/Service Card:** 4:3
- **Team Member:** 1:1
- **Impact/Stats:** Full bleed or 1:1

### Image Treatment
- **Border Radius:** 8px (standard), 0px (full bleed)
- **Overlay:** Dark gradient (when text over image)
- **Optimization:** Compress for web, use WebP where possible

### Photo Direction
- **Authentic & Inclusive:** Farmers, businesses, project scenes
- **Quality:** High-resolution, professional
- **Color Tone:** Warm, vibrant (complements green #61af50)

---

## Motion & Interactions

### Hover Effects
- **Buttons:** Color change + slight scale (1.02)
- **Links:** Color change to #61af50 + underline
- **Cards:** Subtle shadow increase + slight lift (translateY -2px)
- **Images:** Zoom effect (1.05 scale) on container hover

### Transitions
- **Standard:** 300ms ease-in-out
- Applied to: color, background, transform, box-shadow

---

## Notes for Development

- All colors use hex values
- Use CSS variables for consistency (--color-primary, --color-green, etc.)
- Maintain responsive images with picture element and srcset
- Lazy load images below the fold
- Keep font files optimized (only necessary weights/styles)
- Use rem units for scalability (based on 16px base)
